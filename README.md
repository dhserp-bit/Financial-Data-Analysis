# 💊 주요 제약사 유통 비용 구조 비교 분석 (2025.3Q)

## 1. 프로젝트 개요
최근 의약품 유통 대란 및 도매상 난립 이슈와 관련하여, 주요 상장 제약사(**광동제약, 유한양행**)의 유통 관련 비용(운반비, 지급수수료) 구조를 데이터로 비교 분석했습니다.

## 2. 분석 도구 및 방법
* **Language:** Python
* **Libraries:** OpenDartReader, Pandas, Matplotlib
* **Data Source:** * 금융감독원 DART (Open API) : 매출액 자동 수집
    * 2025년 3분기 분기보고서 주석 : 운반비/수수료 정밀 확인 (Hybrid Approach)

## 3. 주요 발견 (Key Findings)
1.  **광동제약의 높은 수수료율:** 광동제약의 지급수수료 비중은 약 **4.53%**로, 유한양행(2.41%) 대비 약 2배 높습니다. 이는 외부 유통망 의존도가 높음을 시사합니다.
2.  **물류비 부담:** 음료(삼다수) 비중이 높은 광동제약이 구조적으로 운반비 부담이 더 큼을 확인했습니다.
