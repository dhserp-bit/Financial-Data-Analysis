# 💊 제약사 유통 비용 구조 및 리스크 분석 (2025.3Q)

## 1. 프로젝트 개요 (Overview)
최근 의약품 도매상 난립과 유통 대란 이슈가 심화됨에 따라, 국내 주요 상장 제약사인 **광동제약**과 **유한양행**의 유통 관련 비용 구조를 데이터로 비교 분석하였습니다.
단순 매출 비교를 넘어, **운반비**와 **지급수수료** 비중을 분석하여 각 기업의 공급망 리스크 노출도를 진단했습니다.

* **분석 대상:** 광동제약 vs 유한양행
* **분석 기간:** 2025년 3분기 (누적 기준)
* **핵심 지표:** 매출액 대비 운반비율(%), 지급수수료율(%)

## 2. 분석 방법 (Methodology) : Hybrid Data Collection
데이터의 신뢰성과 효율성을 동시에 확보하기 위해 **API 자동화와 정밀 수기 검증을 결합한 하이브리드 방식**을 채택했습니다.

1.  **Open DART API 활용 (Python):**
    * 실시간으로 변동되는 핵심 재무 데이터(**매출액**)를 API로 자동 수집하여 분석 파이프라인 구축.
    * API의 `thstrm_add_amount`를 호출하여 기간 불일치 오류(3개월 vs 9개월) 해결.
2.  **Deep Dive (Manual Verification):**
    * 재무제표 본문에 드러나지 않는 **'운반비'**와 **'지급수수료'** 항목은 주석(Footnote)을 직접 확인하여 매핑.
    * 비정형 데이터의 오류 가능성을 차단하고 데이터 정합성 100% 확보.

## 3. 주요 분석 결과 (Key Findings)
데이터 분석 결과, 두 회사는 제품 포트폴리오와 유통 전략 차이로 인해 비용 구조가 명확히 갈렸습니다.

### 📊 1. 지급수수료 (유통망 의존도)
* **광동제약 (4.53%)** vs 유한양행 (2.41%)
* **인사이트:** 광동제약의 수수료 비중이 약 **1.9배** 높습니다. 이는 광동제약이 자체 영업망보다 도매상 등 외부 유통 파트너 의존도가 높음을 의미하며, 유통 시장 혼란 시 수익성 타격 리스크가 더 큼을 시사합니다.

### 🚛 2. 운반비 (물류 부담)
* **광동제약 (1.20%)** vs 유한양행 (1.06%)
* **인사이트:** '삼다수' 등 중량 음료 제품 비중이 높은 광동제약이 구조적으로 물류비 부담을 더 많이 안고 있습니다.

## 4. 사용 기술 (Tech Stack)
* **Language:** Python
* **Data Source:** 금융감독원 DART (Open API)
* **Libraries:**
    * `OpenDartReader`: 공시 데이터 수집
    * `Pandas`: 데이터 전처리 및 비율 계산
    * `Matplotlib`: 데이터 시각화 (한글 폰트 적용)
