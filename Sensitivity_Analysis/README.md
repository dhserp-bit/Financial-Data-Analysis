# 📉 CJ제일제당 원가 민감도 분석 모델 (Cost Sensitivity Analysis)

## 1. 프로젝트 개요 (Overview)
국제 곡물 가격 및 환율 변동성이 심화됨에 따라, 국내 1위 식품 기업인 **CJ제일제당**의 영업이익이 주요 원가 변수에 얼마나 민감하게 반응하는지 시뮬레이션했습니다.
원재료비 상승 시나리오별로 예상 영업이익을 산출하여, **재무적 리스크(Risk Exposure)를 정량화**하는 것이 목표입니다.

* **분석 대상:** CJ제일제당 (식품 사업부문 중심)
* **주요 변수 (X):** 원당(설탕 원료), 소맥(밀가루 원료) 가격, 대두유 가격 등
* **목표 변수 (Y):** 영업이익 (Operating Profit)

## 2. 분석 방법 (Methodology) : Scenario Modeling
단순한 과거 데이터 조회를 넘어, 미래 불확실성에 대비하기 위한 **민감도 분석(Sensitivity Analysis)** 기법을 적용했습니다.

1.  **원가 구조 추정 (Cost Structure Estimation):**
    * DART 사업보고서를 기반으로 매출원가율 및 주요 원재료 매입 비중을 파악.
2.  **시나리오 설정 (Scenario Planning):**
    * **Base Case:** 현상 유지
    * **Best Case:** 원자재 가격 10% 하락 (안정화)
    * **Worst Case:** 원자재 가격 20% 급등 (공급망 충격)
3.  **민감도 계수 산출:** 원재료비가 1% 상승할 때 영업이익이 몇 % 감소하는지 계산.

## 3. 주요 결과 (Key Findings)
*(아래 수치는 분석하신 실제 결과에 맞춰 수정해서 사용하세요)*

### 🚨 1. 원자재 가격 변동 리스크
* 시뮬레이션 결과, 주요 곡물 가격이 **10% 상승**할 경우 영업이익은 약 **O% 감소**하는 것으로 나타났습니다.
* 특히 **설탕(원당)** 가격 변동에 가장 높은 민감도를 보여, 원당 가격 헷징(Hedging) 전략이 필수적임을 확인했습니다.

### 🛡️ 2. 손익분기점(BEP) 방어 전략
* 원가 상승분을 판가(제품 가격)에 **O%** 전가해야만 기존 영업이익률을 방어할 수 있다는 결론을 도출했습니다.
